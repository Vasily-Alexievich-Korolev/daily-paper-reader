## 摘要
大型语言模型遗忘旨在抑制指定的不良知识，同时保留良性能力。许多遗忘目标聚焦于压制不良答案，而最近的目标导向变体指定了替代行为，但仍使更新局部性基本不受约束。本文提出空空间约束响应指定遗忘（NSRU），一种用于受控LLM遗忘的投影约束低秩框架。NSRU使用显式结构化的安全目标响应为每个遗忘查询指定期望行为，同时抑制原始不良内容。为了定位自适应，NSRU从良性的隐藏表示中估计每模块保留子空间，并使用正交投影低秩参数化将LoRA更新限制在保留子空间的零空间中。所得目标在这一约束参数化下联合优化安全目标学习、不良响应抑制和保留保持。我们提供局部一阶分析，表明投影更新减少保留侧扰动，同时保留用于塑造遗忘查询行为的可编辑方向。在TOFU上的实验表明，NSRU有效抑制可提取的遗忘集知识，同时比代表性基线提高保留问答性能、模型效用和安全目标对齐。在WMDP上，NSRU将危险领域准确率保持在接近随机选择区域，同时保留广泛和领域相邻的MMLU效用。消融研究支持安全目标监督、不良响应抑制、保留损失和零空间投影更新的互补作用，而敏感性和鲁棒性分析表明在测试的超参数和提示变化下行为稳定。

## Abstract
Large language model unlearning aims to suppress designated undesirable knowledge while preserving benign capabilities. Many unlearning objectives focus on suppressing undesired answers, while recent target-guided variants specify replacement behavior but still leave update locality largely unconstrained. This paper introduces \emph{Null-Space Constrained Response-Specified Unlearning} (NSRU), a projection-constrained low-rank framework for controlled LLM unlearning. NSRU uses an explicitly structured safe target response to specify the desired behavior for each forget query, while suppressing the original undesired content. To localize adaptation, NSRU estimates per-module retain subspaces from benign hidden representations and uses an orthogonal-projected low-rank parameterization to confine LoRA updates to the null space of the retain subspace. The resulting objective jointly optimizes safe-target learning, undesired-response suppression, and retention preservation under this constrained parameterization. We provide a local first-order analysis showing that the projected update reduces retain-side perturbations while preserving editable directions for shaping forget-query behavior. Experiments on TOFU show that NSRU effectively suppresses extractable forget-set knowledge while improving retain QA performance, model utility, and safe-target alignment over representative baselines. On WMDP, NSRU keeps hazardous-domain accuracy near the random-choice region while preserving broad and domain-adjacent MMLU utility. Ablation studies support the complementary roles of safe-target supervision, undesired-response suppression, retention loss, and null-space projected updates, while sensitivity and robustness analyses indicate stable behavior across the tested hyperparameter and prompt variations.

## 1. 核心问题与整体含义

- **研究动机**：大型语言模型（LLM）的遗忘任务要求模型抑制指定不良知识（如个人隐私、危险指令），同时保留其原有良性能力。现有方法多聚焦于通过梯度上升、偏好优化等目标来压制不良回答，但很少对参数更新的方向施加局部性约束。这种“无约束更新”容易导致模型在遗忘不良行为的同时，也破坏了与保留知识相关的表征方向，造成遗忘 - 保留权衡失控。
- **核心问题**：如何同时做到两点——① 明确指定遗忘后模型应产生的替代行为（响应指定）；② 确保参数更新集中在与保留能力正交的方向上，从而最小化对保留行为的干扰。
- **整体意义**：本文提出 NSRU（Null-Space Constrained Response-Specified Unlearning），首次将响应指定遗忘与保留子空间零空间投影结合在一个统一框架中，为解决 LLM 遗忘中的“更新局部性”问题提供了一种新的约束自适应视角。

## 2. 方法论

- **核心思想**：  
  对于每个遗忘查询，既定义原始不良回答 \(y^-\)，也离线构造一个安全目标回答 \(y^+\)（含安全理由前缀和最终安全应答）；同时从保留集样本的隐藏表征中估计每模块的保留子空间，然后通过正交投影将 LoRA 低秩更新限制在该保留子空间的零空间中，使更新只能作用于与保留方向正交的成分。

- **关键技术细节**：
  1. **响应指定构造**：  
     用外部 LLM 根据遗忘查询和安全提示生成 \(y^+\)，分解为安全理由前缀 \(r^+\) 和最终答案 \(a^+\)。
  2. **保留子空间估计**：  
     对每个选中模块 \(l\)，从保留集样本的隐藏表征中提取特征向量，做无中心 SVD，按能量阈值 \(\rho\) 保留前 \(k_l\) 个奇异方向，得到正交基 \(U_l\)，从而定义保留投影器 \(P_l = U_l U_l^\top\) 和零空间投影器 \(P_l^\perp = I - P_l\)。
  3. **零空间约束低秩自适应**：  
     在冻结的线性层上附加 LoRA 因子，但输入侧先经过 \(P_l^\perp\) 投影：  
     \[
     W'_l h_l = W_l h_l + \frac{\alpha_l}{r_l} B_l A_l (P_l^\perp h_l)
     \]
     仅 \(A_l, B_l\) 可训练，且梯度自然由零空间成分驱动。
  4. **联合优化目标**：  
     \[
     \mathcal{L} = \mathcal{L}_{\text{safe}} + \lambda_f \mathcal{L}_{\text{undesired}} + \lambda_r \mathcal{L}_{\text{ret}}
     \]
     其中 \(\mathcal{L}_{\text{safe}}\) 是 \(y^+\) 的负对数似然，\(\mathcal{L}_{\text{undesired}}\) 是 \(- \ell(y^-)\)（上升），\(\mathcal{L}_{\text{ret}}\) 是保留集负对数似然。
  5. **局部一阶分析**：  
     论文给出理论说明：对于保留表示接近保留子空间的输入，投影更新的影响很小；对于遗忘表示有零空间能量的输入，保留可编辑方向用于行为重定向。

- **算法流程**（文字描述）：
  1. 从保留集中采样子集 \(\tilde{\mathcal{D}}_r\)，对每个选中模块提取特征矩阵，做无中心 SVD 得到保留子空间及零空间投影器，并冻结。
  2. 初始化 LoRA 因子，迭代训练：每个 batch 从遗忘集和保留集采样，计算投影后的 LoRA 前向，分别计算 \(\mathcal{L}_{\text{safe}}\)、\(\mathcal{L}_{\text{undesired}}\)、\(\mathcal{L}_{\text{ret}}\)，反向更新仅作用于 LoRA 因子。
